# Thunderbird_South_Image_Processing
This repository contains data M8(Lagoon Nebula) from UBC's Thunderbird South Telescope, and a jupyter lab file with code to process this data. 

This repo also includes the complete solution code for the workshop which inlcudes code that is run to precalibrate the fits data from the telescope

There are big files on this repo. To clone this repo you need to import git large file storage(lfs).
Run the folowing code in your terminal
based on your operating system

<u> On MacOS </u>
```
brew install git-lfs
```

<u> On Windows:
Download from git-lfs.github.com or use: </u>
```
winget install git-lfs
```

<u>On Linux:</u>
```
sudo apt-get install git-lfs
```
